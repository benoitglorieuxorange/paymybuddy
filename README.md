
# PayMyBuddy


## Description

PayMybuddy est une application de transfert d'argent entre amis
Seuls les "amis" précédemment ajoutés (via leur adresse email) pourront recevoir de l'argent.
La liste des transactions effectuées apparaît en page d'accueil.

Un formulaire d'enregistrement permet de s'inscrire sur le site payMyBuddy

L'utilisateur peut changer son mot de passe dans la page profil
Le solde de la personne connectée apparaît sur la page d'accueil et dans son profil.
- il n'est pas possible de supprimer des transactions,
- Si un utilisateur est supprimé, ses transactions restent apparentes sur la page d'accueil des personnes à qui il a envoyé de l'argent.
- Par mesure de sécurité si la transaction ne se passe pas normalement (enregistré dans la bdd), ni le solde de l'émetteur ni celui du destinataire ne sont modifiés.
