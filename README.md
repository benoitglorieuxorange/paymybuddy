
# PayMyBuddy


## Description

PayMybuddy est une application de transferty d'argent entre amis
Seul les "amis" précédement ajouter (via leur adresse email) pourront recevoir de l'argent.
La liste des transactions effectuées apparait en page d'accueil.

Un formulaire d'enregistrement permet de s'inscrire sur le site payMyBuddy

L'utilisateur peut changer sont mot de passe dans la page profil
Le solde de la personne connectée apparait sur la page d'accueil et dans son profil.
- il n'est pas possible de supprimer des transaction,
- Si un utilisateur est suppérimé ses transaction restent apparant sur la page d'acceuil des person a qui il a envoyé de l'argent.
- Par mesure de securité si la transaction ne se passe pas normalement (enregistré dans la bdd), ni le solde de l'emetteur ni celui du destinataire ne sont modifié.
